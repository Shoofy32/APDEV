# Blevvit

<img width="997" height="211" alt="Blevvit_Logo" src="https://github.com/user-attachments/assets/b0520f79-e10c-4287-9c15-65fd9a454e92" />


<b> Blevvit (working title) is a WIP online forum meant to facilitate small yet passionate discussions regarding whatever a user wishes to discuss. It is modern, lightweight, and moderated. Unlike other forums, it has a challenge feature, making gaining likes fun and interactive. </b>

---

## Current Features

- Home Page with sidebar and search bar
- Various forums and topic pages
- Simple Log in and registraction page and system (No backend)
- Search system with filtering by tags
- Post and reply system for forum pages
- Challenge feature by betting likes and rolling a D20
- Slight backend for posts
